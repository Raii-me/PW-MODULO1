1. O que é um Banco de Dados
2. Tipos de Banco de Dados

  • Banco de Dados (BD) é um conjunto organizado de informações que podem ser facilmente acessadas, gerenciadas e atualizadas. Ele é usado para armazenar dados de forma estruturada, como nomes, senhas, produtos, vendas, notas de alunos, entre outros.
  • Relacional: organiza os dados em tabelas que se relacionam entre si. Exemplo: MySQL, PostgreSQL, Oracle.
  • Não-relacional (NoSQL): trabalha com dados mais flexíveis, como documentos e coleções. Exemplo: MongoDB, Firebase.
  • Local: instalado e acessado em apenas um computador.
  • Na nuvem: acessado pela internet, como o Google Cloud ou Amazon AWS.


3. SGBD (Sistema de Gerenciamento de Banco de Dados)
4. Importância dos Bancos de Dados

  • SGBD é o software que permite criar, manter e manipular os dados de um banco de dados. Ele garante que os dados sejam armazenados com segurança, organização e eficiência.
  • Exemplos de SGBD: MySQL, SQL Server, SQLite, MongoDB.
  • Armazenam grandes volumes de dados com segurança.
  • Permitem consultas rápidas e personalizadas.
  • São usados em praticamente tudo: redes sociais, lojas virtuais, escolas, hospitais, bancos, etc.
