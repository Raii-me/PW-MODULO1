1. O que é Programação
2. O que é um Algoritmo

  • Programação é o processo de escrever instruções (código) que um computador pode entender e executar. Ela é usada para criar sites, aplicativos, sistemas, jogos e muito mais. As linguagens de programação (como Python, JavaScript ou C++) têm regras próprias, chamadas de sintaxe.
  • Algoritmo é uma sequência de passos lógicos e bem definidos para resolver um problema ou executar uma tarefa. Ele pode ser escrito em linguagem natural, pseudocódigo ou até representado por fluxogramas. Exemplo: um algoritmo para preparar um café seria listar os passos desde esquentar a água até servir na xícara.


3. Como Programação e Design se conectam
4. Ferramentas e Tecnologias Usadas

  • Apesar de parecerem áreas diferentes, design e programação andam juntos:
  • O design cria a aparência e experiência (UX/UI).
  • A programação transforma o visual em algo funcional. Por exemplo, o designer monta o layout de um site, e o programador implementa isso com HTML, CSS, e JavaScript.
  • Design: Figma, Photoshop, Illustrator, Canva, Adobe XD.
  • Programação web: HTML, CSS, JavaScript, React, Node.js.
  • Trabalhos modernos costumam usar frameworks que unem os dois, como React (programação) + Figma (design) para criar interfaces completas.
