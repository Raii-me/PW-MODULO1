1. O que é PW – Programação Web
2. Divisões da Programação Web

  • PW, ou Programação Web, é a área da informática responsável pela criação de sites, sistemas e aplicativos para a internet. Envolve tanto o que o usuário vê (visual) quanto o que acontece por trás (funcionamento e lógica). É uma das áreas mais importantes do desenvolvimento digital atualmente.
  • Front-end: parte visual da aplicação. Usa linguagens como HTML, CSS e JavaScript. É o que o usuário vê e interage.
  • Back-end: parte lógica e de funcionamento. Usa linguagens como PHP, Python, Node.js, entre outras.
  • Full Stack: profissional que trabalha com as duas partes (front e back-end).



    3. Tecnologias e Ferramentas Usadas
    4. Importância da Programação Web

    • HTML: estrutura da página (títulos, textos, imagens).
    • CSS: estilo visual (cores, fontes, layouts).
    • JavaScript: interatividade (menus, animações, formulários).
    • Frameworks: como Bootstrap (CSS), React (JS), Laravel (PHP), entre outros.
    • Banco de dados: como MySQL ou Firebase, usados para armazenar informações.
    • Permite a criação de sites, lojas virtuais, blogs, redes sociais e muito mais.
    • Está presente em quase tudo na internet.
    • É uma habilidade muito procurada no mercado de trabalho.
    • Desenvolve o pensamento lógico e criativo dos alunos.
